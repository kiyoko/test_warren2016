Instractions to Submit Abstract

  • 1. Register to attend Warren Workshop VI and make payment before submitting abstract.
          If you have not registered or made payment, please go to registration page.

  • 2. Please use the downloadable MS Word file as a template and prepare abstract.


  • 3. Name the file as following;
     • If your abstract is for oral presentation, save file as       “Presenter’sLastnameFirstname_oral.docx”
     • If your abstract is for poster session, save file as
           “Presenter’sLastnameFirstname_poster.docx”
     • If your abstract is for bioinformatics hands-on session, save file as
          “Presenter’sLastnameFirstname_bio.docx”

  • 4. Convert file extension from WORD (finename.docx) to PDF (finename.pdf) before submit the
         abstract.

  • 5. Email the abstract to the following email address: No later than June 30, 2016.
     Indicate the following information with your submission via email:
          • Your full name
         • Affiliation
          • Title of abstract
         • Full name of the presenting author
         • Email address of the presenting author
         • Your preference on the mode of presentation
         -oral, poster, or bioinformatics hands-on session
         • Date you would like your abstract to be considered for presentation.
